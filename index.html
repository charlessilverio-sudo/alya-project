Sincronização GitHub
Dados compartilhados em tempo real via repositório GitHub
Como funciona: os dados são salvos em um arquivo data.json no seu repositório GitHub. Qualquer pessoa com o link do app vê os dados atualizados. Só quem tem o token pode salvar.
Deixe data.json na raiz do repositório
⚠️ O token fica salvo apenas no seu navegador (localStorage). Nunca é enviado a terceiros — só ao GitHub.
📋 Como criar o token (passo a passo)
1. Acesse github.com/settings/tokens/new
2. Dê um nome como alya-project
3. Em Expiration, escolha No expiration
4. Em Select scopes, marque apenas repo
5. Clique em Generate token
6. Copie o token (começa com ghp_) e cole acima
